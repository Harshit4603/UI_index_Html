# SalesAssist - AI Sales Chatbot

A beautiful, modern frontend for your Sales Chatbot.

## Features
- Login as Sales Rep or Admin
- Sliding chat history sidebar
- Message copy with toast notification
- Prev/Next message navigation
- LocalStorage persistence per role
- Clean UI with Tailwind CSS

## How to Run Locally
1. Just open `index.html` in any browser (Chrome, Edge, Firefox).
2. No installation needed.
